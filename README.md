# Customized Subscription Tracker

A full-stack web application that helps users **track their subscriptions**, receive **timely email reminders**, and make informed decisions about **renewing or canceling services**.

---

# Objective

Many users forget about recurring subscriptions and get charged unintentionally.  
This project solves that problem by:

- Tracking all user subscriptions in one place  
- Sending automated reminder emails before renewal dates  
- Helping users take action on time
